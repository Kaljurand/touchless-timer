Timer

Timer is an alarm clock and timer application that uses speech recognition to offer a hands-free user interface. Timer understands two types of speech commands: timer commands specify a time period in hours and minutes from the current time, and clock commands specify the 24h clock time in hours and minutes, e.g.

  • wake me up in six minutes from now
  • wake me up in two hours and thirty minutes from now
  • set the alarm to eighteen oh three
  • set the alarm to twenty three fifty nine

The effectiveness of a speech recognition based user interface depends on your distance from the microphone, the quality of the microphone and the presence of background noise. These conditions also determine the suitable use cases for this application (e.g. kitchen timer, night desk alarm clock, etc.).

To stop the timer, tell it the current time, or simply close the browser tab/window.

Installation

Timer is a web application, i.e. it runs in a web browser, either on your desktop, tablet, or smart phone. This makes its installation and usage straight-forward, just click on /timer/en.html and bookmark it. Timer runs completely offline, i.e. does not require the internet connection after installation. Note that Timer requires a very modern web browser (with support for WebRTC, AppCache, fast Javascript engine, etc.). We have tested it only with the very latest version of Google Chrome.

Used technologies

Timer is based on the technologies of pocketsphinx.js, meSpeak.js, and Grammatical Framework. The grammar of the input language is formally specified in the Grammars-project, some randomly generated examples are here.
